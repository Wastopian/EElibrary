# UI / UX Brief

## Product feel
- industrial
- trustworthy
- dark-mode friendly
- data-dense without being ugly
- fast to scan

## Main pages
### Dashboard
- global search
- recent parts
- saved filters
- quick tools
- library health

### Search
- filter rail
- results list/table toggle
- asset badges
- compare selection

### Component detail
- MPN hero
- specs table
- package dimensions
- datasheet panel
- footprint / symbol / 3D asset cards
- trust / validation badges
- export drawer

### Compare
- key specs
- package differences
- CAD completeness
- lifecycle / trust

### Tools
- calculators
- cross-reference tools
- BOM helpers

### Admin
- ingest queue
- conflicts
- failed parses
- approvals

## Visual direction
- graphite/slate background
- electric blue accents
- muted green for verified
- amber for review
- mono font for MPNs and filenames
